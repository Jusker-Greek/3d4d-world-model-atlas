02 · THE CENTRAL QUESTION

“学到了 3D”有四个不同含义

这四层经常在论文叙事中混用。判断是否形成 3D 分布，关键不是训练样本多，而是模型是否定义并验证了同一条件下多个合理世界。

01
PER-SCENE OPTIMIZATION

单场景拟合

θᵢ* = argmin L(Render(θᵢ), Iᵢ)

NeRF / 3DGS 常见范式。每个场景独立优化一个参数实例；跨场景没有共享生成分布。

02
AMORTIZED PRIOR

跨场景确定性先验

Ŝ = fθ(I)

DUSt3R、VGGT、D4RT、Point4Cast 的主体。数据集把规律压进权重，但同一输入通常只回归一个答案。

03
EXPLICIT CONDITIONAL DISTRIBUTION

可采样的条件分布

S ~ pθ(S | I, A)

MVDream、TRELLIS、O-Voxel、WorldReel、U4D、3WM。必须看多次采样、分布指标或校准，而非只看单次重建误差。

04
PERSISTENT BELIEF

随观测更新的世界 belief

bₜ(S) ∝ p(oₜ | S, aₜ)·bₜ₋₁(S)

P3Sim 最接近这一定义；CUT3R/Point4Cast 有持久状态，但状态仍是确定性 token，不自动等于 belief distribution。

一句话判断

“跨很多 3D 样本训练”只证明学到 prior；只有对固定条件能产生并评估多个合理 3D/4D 假设，才证明学到 distribution。

03 · FALSIFICATION PROTOCOL

Pixel→World→Pixel：如何排除强 autoencoder 捷径

这组实验不能从哲学上“证明模型理解了世界”；它能够逐层否定更弱的解释：纹理匹配、训练轨迹插值、成对像素回归和无物理含义的 token cache。

首先修正一个标准

“知道这是椅子”和“知道相机转 30° 时表面点怎样投影”是两种能力。一个 pose 模型可以有真实的投影几何能力而没有类别语义；因此应在论文的声称边界内分层测试。

GGEOMETRIC

投影几何

depth、pose、world point、visibility 可以在不看目标像素时预测目标位置，并满足 SE(3) 组合。

OOBJECT-CENTRIC

对象持久性

纹理、视角和遮挡改变后，仍保持同一对象、部件、身份和 3D 轨迹。

PPHYSICAL

物理因果

可以分别介入 camera、object motion、light、material 或 action，而不污染无关状态。

DDISTRIBUTIONAL

条件分布

相同可见证据存在多个隐藏世界/未来时，输出覆盖各 mode 并随新证据收缩。

可解释路径
pixel p→depth + K + T→world point X→K′ + T′ + visibility→pixel p′
X = T⁻¹ π⁻¹(p,d,K)  ;  p′ = π(K′T′X)
不能被普通重建 loss 排除的路径
input pixels + camera token→high-dimensional latent→target pixels / pointmap

它在 IID 数据上可以一样准；只有因子介入、组合和闭环才能区分两者。

每次只改一个物理因子；不把换纹理、换相机和换数据域捆在同一次实验里。

源像素能否先经由可解释的深度、相机和世界点，再投影到目标像素？

两次相机/对象变换是否等于一次复合变换，反变换是否回到原状态？

F(g₂, F(g₁, W)) ≈ F(g₂g₁, W)

只改相机、几何、纹理、光照或动作中的一项，其他世界变量会不会被污染？

do(zⱼ ← zⱼ′), Δzⱼ ≠ 0,  Δzₙ₋ⱼ ≈ 0

IDₜ₋₁ = IDₜ₊ₖ, Xₜ₋₁ → Xₜ₊ₖ

DIST-7

歧义条件下的多世界分布

完全相同的可见证据对应多个隐藏 3D/未来时，模型能否覆盖而不是取平均？

W⁽¹⁾, W⁽²⁾ ~ p(W | I), R(W⁽ⁱ⁾) ≈ I
ACT-8

动作条件的局部因果

同一初始世界只换一个动作，预测是否只改变受作用物体并保持其他状态？

MINIMAL GATE LADDER

第一轮先不训新模型

  1. H1a · 纹理介入

    只替换贴图，冻结模型、mesh、camera、light 和 metric。如果 pose/pointmap 大幅改变，先拒绝几何不变性。

  2. H1b · 相机组合

    H1a 通过后，只改 camera transform；检查 g₁ 后 g₂ 是否等于 g₂g₁，以及 A→B 是否与 B→A 互逆。

  3. H1c · 中介因果

    禁用 learned pixel decoder，只把模型预测的 W = {depth, pose, points, visibility} 送入固定解析 renderer。若目标视图仍可解释，才证明 W 对任务是充分中介。

对照组

同一评测同时放入纯 pixel predictor、经典 calibrated triangulation/BA 和 GT geometry renderer。前者是“强 autoencoder”对照，后两者给出几何规律和上限。

PRIOR ART

已有评测到哪一步

这个方向不是从零开始；新的部分是把 feature probing、解析 PWP 闭环、因果介入和歧义分布合成对 3D/4D GFM 的分层协议。

I/O 变化
(I₁, masked I₂) → RGB patches of I₂
因此新增

DUSt3R

改动位置
监督信号与稠密输出头

(I₁, I₂) → (X¹¹, X²¹, confidence)
因此新增

仍未解决

最小增量

因此新增
把“有几何”扩展到“能稠密找到对应”，改善宽基线匹配和对齐。
仍未解决
没有把 pair-wise 模型变成全局场景模型，也不解决动态或度量尺度。

改动位置
主干的视图聚合方式
最小增量

image pairs + alignment → an image set → joint global geometry
因此新增
1–数百视图的单次前馈全局重建，相机与几何共享同一 context。
仍未解决

06FROM · VGGT 架构/权重直接继承

π³

改动位置
参考坐标系和视图顺序处理
最小增量

I/O 变化
ordered views in one anchor → an unordered/equivariant view set
因此新增

仍未解决

07FROM · SRT-style query decoding + video pretraining问题延续

D4RT

改动位置
输出接口和 decoder 形式
最小增量

fixed dense heads → sparse 4D ray queries over one video latent
因此新增
同一 decoder 同时表达重建、跨时对应和不同参考系，且稀疏查询无需解码全帧。
仍未解决

改动位置

reconstruct or track separately → reconstruction + long-range 3D tracking together

pair representation 到 video backbone 的扩展
最小增量

two-frame DPM → multi-frame video DPM queried by target time
因此新增

DynamicVGGT

static global geometry heads → current/future points + Gaussian velocity
因此新增
把全局重建扩展到短 clip 的 4D reconstruction 和短期 forecast。

Any4D

改动位置

最小增量

07FROM · persistent-state GFM + dynamic pointmaps问题合并

Point4Cast

改动位置
视频处理方式与时间读出域
最小增量
将整段 batch encoder 换成每帧更新的固定容量 spacetime tokens；readout 同时接收 query frame 和 query time，可读任意已观测过去/当前，也读未观测未来 pointmap。
I/O 变化
offline observed-video reconstruction → streaming state + arbitrary-time readout
因此新增

C · 持久记忆

不再把每个 image pair 当独立样本；分别将历史放入外部 memory、递归 latent state 或时空 state。

改动位置

independent image pairs → frame + external spatial memory → world pointmap

因此新增

spatial persistent state → time-indexed spacetime state
因此新增
记忆不再只用于当前重建，而成为可用于 4D 轨迹与 forecast 的统一中介。
仍未解决

E · 原生 3D 生成分布
